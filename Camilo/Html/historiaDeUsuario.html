Historia de Usuario

Como estudiante de programación, quiero desarrollar una página web personal que permita mostrar información sobre mí, mis conocimientos y mis objetivos,para practicar HTML, CSS y la organización de un sitio web.

Necesidad

Necesito una página sencilla, ordenada y fácil de navegar donde cualquier persona pueda conocer quién soy y cuáles son mis habilidades.

Resultado esperado

Al finalizar el proyecto, tendré un sitio web conectado entre varias páginas, con un diseño agradable y organizado utilizando HTML y CSS.
